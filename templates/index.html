Patient Demographics Datavis

This page provides an outline of active (alive) patients' demographics. It is intended for use by healthcare data analysts to gain insights into the patient population.

Submit a patient record

Available API Endpoints

These API endpoints provide access to the available patient records:

Method Endpoint Description
GET /api/patients/{id} Get details of a specific patient
GET /api/patients Get a list of all patients
POST /api/patients Submit a new patient to the database
Required fields: 
{
"id": string,
"gender": "male" | "female",
"birthDate": date
}
